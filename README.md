# Campus Energy Dashboard

This project analyzes campus electricity consumption across multiple buildings using Python.  
It reads meter data, cleans it, generates summaries, and visualizes trends through charts.

### Features
- Data loading and validation
- Building-wise and time-based energy analysis
- OOP-based structure for handling buildings and readings
- Dashboard visualization and text summary export

### How to Run
1. Place CSV data files in the `data/` folder.
2. Run the main script.
3. View outputs in the `output/` folder (plots, summary, cleaned data).

### Purpose
Created as a capstone project demonstrating Python file handling, data analysis, visualization, and OOP concepts.
